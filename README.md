# 杀戮尖塔2：储君保留铸剑 | STS-2-Regent-Keep-Forge

[![Game](https://img.shields.io/badge/Game-Slay%20the%20Spire%202-blue)](https://store.steampowered.com/app/2868840/Slay_the_Spire_2/)
[![Platform](https://img.shields.io/badge/Platform-Steam-informational)](https://store.steampowered.com/app/2868840/Slay_the_Spire_2/)

这是一个为《杀戮尖塔2》（Slay the Spire 2）设计的娱乐 Mod。

### 📖 简介 | Introduction

字面意思，由于铸剑过于猪鼻，本 Mod 修改了铸剑的内容，使得**储君的铸剑效果可以跨战斗保留**。支持多人联机。

Literally as the title says. Since the vanilla sword forging is a bit "underwhelming," this mod allows **The Regent's Sword Forging effects to persist across battles**. Support multiplayer online.

---

### 🛠️ 安装方法 | Installation

1. 安装了《杀戮尖塔2》
2. 下载本项目发布的压缩包
3. 将压缩包解压至游戏的 `mods` 文件夹内
4. 启动游戏
